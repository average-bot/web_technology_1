Delfrågor:

  1. Vad menas med kaskad i CSS? Hur fungerar det?
  2. I HTML och CSS har vi vidstående kod. Detta resulterar i två boxar. Vad blir de markerade avstånden a och b i pixlar? Beskriv också hur a beräknas.

Mitt svar

Under de nedstående kapitlarna kommer delfrågorna E1 och E2 att besvaras.

E1

Kaskad i CSS betyder att stilarna kan falla från en stilark till en annan. Det vill säga att man kan ha flera stilarkar till ett HTML-dokument.

E2

  1. Först ska man räkna ut padding i pixlar =>
    padding = 1.5em = 1,5*(font-size) = 1,5*12px = 18px
    Sedan kan man räkna ut a med hjälp av att addrera ihop värderna för 2 border, 2 padding och width =>
    2*border + 2*padding + width = 2*4px +2*18px + 120px = 164 px
    a = 164px
  2. Marginalerna överlappar varandra, det vill säga att om båda boxarnas marginal är 20px då är mellanrummet mellan boxarna 20px på grund av att marginalerna överlappar varandra.
    b = 20px.
